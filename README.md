## About the project

> The client source code of the Animify online streamer

### Built with
- React JS
- React Router
- Bootstrap
- Axios
- qs

## Getting Started

### Installation

_Clone the project repo_

    $ git clone https://github.com/tarek99samy/Animify-Frontend.git

_Navigate to project folder_

    cd Animify-Frontend/

_Install project dependencies_
    
    npm i

### Run for development
    npm start

### Run for production
    npm run build
